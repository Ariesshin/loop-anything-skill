Product

What it does

Loop Anything is a skill that turns an AI first draft into a final draft. It adds a review loop around the work, with each reviewer checking the same draft from a different angle.

Review from more than one angle

The run uses 2-3 isolated AI reviewers so one reviewer can protect the reader, another can protect feasibility, and another can protect evidence or risk.

Revise until objections are gone

Reviewers critique the work, the main agent revises, and the loop continues until every reviewer has nothing left to object to.

Use 120 as the stop signal

Score: 120 means that reviewer has zero remaining reservations. Anything below 120 means revise again.

Workflow

How it works

  1. 1

    Start with a draft or request

    Ask your agent to use Loop Anything on a deliverable that deserves extra review.

  2. 2

    Create isolated reviewers

    The agent selects 2-3 review angles and sends bounded review packets to isolated reviewers.

  3. 3

    Revise from the objections

    The main agent synthesizes the objections, updates the draft, and sends the new version back through review.

  4. 4

    Stop only when reviewers pass

    The loop stops when every reviewer returns PASS with Score: 120, or when the run is honestly marked as not fully passed.

Use cases

Use cases

Loop Anything is for the important 5% of outputs where being wrong is costly.

Specs and plans

Use it when a plan needs intent checks, operational realism, and clear next actions.

Proposals and difficult messages

Use it when audience reaction, wording, evidence, and risk all matter.

Prompts and decisions

Use it when a system prompt, job application, or decision memo needs more than a quick draft.

Tradeoff

Tradeoff disclosure

Loop Anything trades more time and estimated visible tokens for higher confidence. It is not meant for every quick response. Use it when review quality matters more than speed.
